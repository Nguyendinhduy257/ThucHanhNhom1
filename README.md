-  Hướng dẫn sử dụng: 
B1: Copy Url của repository này


B2: Tạo thư mục "XXX" để chuẩn bị chứa repository


B3: truy cập CMD trong "XXX" và gõ lệnh "git clone <Url của repository này>"


B4: Mở Visual Studio Code và chọn thư mục vừa CLONE về


B5: gõ "Ctrl + ~" và ghi "cd ThucHanhNhom1"


B6: gõ "npm run web" (chạy trên Web) hoặc "npx expo start" (chạy trên ứng dụng "Expo Go" của điện thoại)


B7: Tận hưởng sản phảm \(O.O)/
